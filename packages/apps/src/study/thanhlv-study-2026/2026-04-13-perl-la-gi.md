# Perl là gì?
Perl là một ngôn ngữ lập trình thông dịch(interpreted language) được tạo vào năm 1987.

Perl thường được sử dụng để xử lý văn bản, tự động hoá và viết script nhanh.

## Đặc điểm chính của perl
- Cực mạnh về xử lý String, text. Bởi vì Perl rất mạnh khi sử dụng Regex. Regex làm nên sự nổi tiếng của perl.
- Viết script nhanh: Phù hợp với các script, tool nội bộ.

## Perl thường dùng để làm gì?
- Xử lý log, file, dữ liệu text.
- Viết script tự động, job.
- Web backend(Năm 2026 thì không còn phổ biến, điều này khá phổ biến ở năm 2014)
- DevOps / System admin

## Ví dụ code perl
```perl
#!/usr/bin/perl
use strict;
use warnings;

my $name = "Thanh";
print "Hello, $name\n";
```

## So với ngôn ngữ khác
- Giống Python: đều dùng để viết script nhanh
- Nhưng Perl:
  - Khó đọc hơn 😅
  - Mạnh hơn về regex
  - Ít phổ biến hơn hiện nay

- Do nhu cầu công việc cần học.
